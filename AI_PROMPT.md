## GitHub 社交预览图尺寸

上传封面图到 GitHub 时：
- **推荐尺寸**: 1280 × 640 px
- **格式**: PNG 或 JPG
- **文件大小**: 小于 1MB

上传路径：`GitHub 仓库 → Settings → Options → Social Preview`
